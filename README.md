Demo: https://wnal4634.github.io/weather_api


공공데이터 API를 사용해 서울시 실시간 날씨와 대기오염정보를 알 수 있는 웹페이지입니다.

정보는 각 행정구별로 확인할 수 있으며, 날씨와 더불어 해당 기온에 맞는 옷 종류를 추천합니다. 또한 비나 눈이 오는 경우에는 우산을, 미세먼지 수치가 높은 경우에는 마스크를 제안합니다.
